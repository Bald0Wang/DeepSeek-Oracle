# Unified Response Contract

Compose final answer in this order:

1. 安抚一句话
2. 核心解读
3. 可执行建议（2-4 条）
4. 风险提示（按安全等级）
5. 可追问方向（3 条）

Action item format:
- `task`: clear and finishable
- `when`: suggested time window
- `reason`: one-line rationale

Disclaimer level:
- `none`: regular low-risk consultation
- `light`: important life choices
- `strong`: finance, medical, legal, or potential harm
